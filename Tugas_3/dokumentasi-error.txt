### Error 1 â€” Tag DOCTYPE Tidak Benar

| Kategori | Detail |

### Error 5 â€” `header()` Setelah Output (Kemungkinan "Headers Already Sent")

| Kategori | Detail |
| :--- | :--- |
| **Pesan Error** | `Warning: Cannot modify header information - headers already sent by (output started at ...)` |

| **Perbaikan** | Pastikan **tidak ada *output* sama sekali** sebelum *header* PHP dan simpan file dalam *encoding* **tanpa BOM** (misalnya UTF-8 tanpa BOM). |
